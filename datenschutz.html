Zweck der Datenverarbeitung

Die Plattform VergabeAssist verarbeitet personenbezogene Daten ausschließlich zum Zweck der Bereitstellung von KI-gestützten Tools zur Erstellung von Vergabeunterlagen für öffentliche Auftraggeber.

Verarbeitete Daten

  • Registrierungsdaten (Name, Organisation, dienstliche E-Mail, optional Telefon)
  • Projektdaten (Leistungsbeschreibungen, Kriterien etc.)
  • Nutzungsdaten (Logins, Zugriffe, Fehler-Logs)

Datenverarbeitung durch Dritte / KI-Provider

KI-Anfragen werden ausschließlich über Server in der EU verarbeitet (Hosting: IONOS, Supabase EU). Für KI-Funktionen werden Anfragen an OpenRouter/OpenAI gesendet. Es werden keine personenbezogenen Daten an KI-Provider übermittelt.

Speicherdauer & Löschung

Alle personenbezogenen Daten können im Accountbereich jederzeit gelöscht werden. Nach Account-Löschung werden die Daten umgehend entfernt.

Betroffenenrechte

Sie haben jederzeit das Recht auf Auskunft, Berichtigung, Löschung und Einschränkung der Verarbeitung Ihrer Daten. Kontakt: datenschutz@govdisrupt.de

AV-Vertrag

Ein Auftragsverarbeitungsvertrag gemäß Art. 28 DSGVO ist Bestandteil der Registrierung und wird im Folgenden bereitgestellt.

Weitere Hinweise

Es dürfen keine personenbezogenen Daten (insbesondere von Dritten, wie Bieter, Schüler, Patienten etc.) über die Plattform hochgeladen oder verarbeitet werden.

Hier geht es zum Auftragsverarbeitungsvertrag (AVV).
